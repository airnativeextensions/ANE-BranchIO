informationsproperty
informations:String  [read-only]

informations contains the error from the base SDK or the the result expected like for getShortUrl, getReferralCode, createReferralCode. It can also contains credits, think to turn them into int! And also the credits history as a stringified array! Or the code referral applied as stringified JSON.
